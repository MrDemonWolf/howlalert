# FAQ

**Does HowlAlert read my Claude conversations?**
No. HowlAlert only reads token count metadata from session files, never the content of your conversations.

**Does it work with Claude Free?**
No. Claude Free has no fixed token limit that HowlAlert can monitor. Supported plans: Pro, Max 5, Max 20.

**Do I need a Claude account in HowlAlert?**
No. HowlAlert reads your existing Claude Code credentials locally.

**What happens if I uninstall?**
All data is removed. The APNs token in CloudKit is deleted when you uninstall iOS. Nothing persists server-side.

**Is there a family/team plan?**
Not currently. Each device requires its own subscription.

**Does it work on Apple Silicon and Intel Macs?**
Yes, HowlAlert is a universal binary supporting both architectures.
