> **TL;DR** — Install the macOS app, grant access to Claude Code's data directory, install the iOS app, and you're done.

## Prerequisites

- macOS 15 Sequoia or later
- Claude Code installed and in use
- iPhone with iOS 18 or later (for push notifications)
- Apple Watch with watchOS 11 or later (optional)

## Quick start

1. [Install the apps](/docs/getting-started/installation)
2. [Configure thresholds](/docs/getting-started/configuration)
3. Start using Claude Code — HowlAlert will notify you automatically
