> **TL;DR** — Set daily cost and token thresholds in the menu bar popover. Notifications fire automatically when thresholds are crossed.

## Usage thresholds

Open the HowlAlert menu bar popover and click **Settings** to configure:

| Setting | Default | Description |
|---------|---------|-------------|
| Daily cost alert | $5.00 | Alert when daily spend reaches this amount |
| Monthly cost alert | $50.00 | Alert when monthly spend reaches this amount |
| Token rate alert | 100k/hr | Alert when hourly token rate exceeds this |

## Notification types

- **Threshold reached** — Fires once when you cross a configured limit
- **Daily summary** — Optional digest at a time you choose
- **Session end** — Fires when Claude Code session ends with cost summary
