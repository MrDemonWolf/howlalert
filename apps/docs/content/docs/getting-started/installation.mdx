> **TL;DR** — Download from the App Store or build from source.

## macOS

Download **HowlAlert** from the [Mac App Store](https://apps.apple.com/us/app/howlalert/id6760729438) or build from source:

```bash
git clone https://github.com/mrdemonwolf/howlalert.git
cd howlalert
make build-mac
```

After launch, HowlAlert appears in your menu bar as a bell icon.

## iOS

Install **HowlAlert** from the [App Store](https://apps.apple.com/us/app/howlalert/id6760729438) on your iPhone. The watchOS app installs automatically via Watch app pairing.

## First launch

On first launch, HowlAlert will ask permission to access Claude Code's data directory (`~/.claude/`). Grant this access to enable usage monitoring.
