# Getting Started

## Requirements

- macOS 14+ (Sonoma or later)
- iOS 17+ (for iPhone companion)
- watchOS 10+ (optional, for Watch complication)
- Active Claude Code subscription (Pro, Max 5, or Max 20)

## Installation

### macOS

1. Download HowlAlert from the Mac App Store
2. Open HowlAlert — it appears in your menu bar
3. Grant file access when prompted (needed to read `~/.claude/` session files)

### iPhone

1. Download HowlAlert from the iOS App Store
2. Open the app — it will register for push notifications
3. Pairing happens automatically via iCloud (no account needed)

## Configuration

Open HowlAlert preferences on macOS:
- **Warn threshold:** Alert when usage reaches X% (default: 75%)
- **Critical threshold:** Alert when usage reaches X% (default: 90%)
- **Paired devices:** Shows all paired iOS devices

## Troubleshooting

**No notifications received?**
- Ensure both apps are signed into the same iCloud account
- Check notification permissions in iOS Settings → HowlAlert
- Verify macOS has file access in System Settings → Privacy & Security → Files and Folders
