# Getting Started

HowlAlert monitors your Claude Code usage in real-time and sends push notifications to your iPhone and Apple Watch when you're approaching your limits.

## How It Works

1. **macOS Menu Bar App** watches your Claude Code session files
2. **Cloudflare Worker** relays usage alerts via APNs
3. **iOS/watchOS** receives push notifications with your usage status

## Installation

### macOS (Homebrew)

```bash
brew install --cask mrdemonwolf/tap/howlalert
```

### iOS & watchOS

Download from the [App Store](https://apps.apple.com/app/howlalert).

## Setup

1. Install the macOS app and the iOS app
2. Sign in to the same iCloud account on both devices
3. Pairing happens automatically via CloudKit
4. You'll start receiving usage alerts on your iPhone and Apple Watch
