# HowlAlert

HowlAlert monitors your Claude Code token usage in real-time and sends push notifications to your Apple devices before you hit your plan limit.

## Quick Start

1. Install HowlAlert on your Mac from the [App Store](#)
2. Install HowlAlert on your iPhone
3. Both apps auto-pair via your iCloud account (no sign-in needed)
4. Receive alerts at 75% and 90% of your Claude plan limit

## Supported Plans

| Plan | Token Limit (5h window) |
|------|------------------------|
| Pro | 32,000 |
| Max 5 | 160,000 |
| Max 20 | 640,000 |

## Architecture

HowlAlert uses a stateless Cloudflare Worker as a push relay. No personal data is stored server-side. Device tokens are synced via CloudKit's private database.
