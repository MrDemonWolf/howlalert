# Multi-Mac Setup

## Why It Matters

Claude Code rate limits are **per Anthropic account, not per machine**. If you run Claude on your MacBook Pro and your Mac mini, usage sums toward one shared limit.

HowlAlert aggregates usage across all your Macs so pace math is accurate.

## How It Works

1. Each Mac writes its own `UsageSnapshot` to CloudKit (keyed by device ID)
2. Your iPhone subscribes to snapshot changes and aggregates them
3. Threshold notifications are based on **combined** usage
4. Notifications include which Mac crossed the threshold

## Setting Up

There's nothing to configure. Just install HowlAlert on each Mac signed into the same Apple ID. Each Mac auto-registers in CloudKit.

## Dashboard

When multiple Macs are paired, the iOS dashboard shows an "Active Macs" card:

- Each Mac listed with name, device type icon, last-seen time, and contribution percentage
- Tap to expand the full list
- Macs that haven't updated in 30+ minutes are marked "idle" (usage still counted)

When only one Mac is paired, this card is hidden.

## Notifications

With multiple Macs, notifications include the device name:

> MacBook Pro · 85% used — runs out in ~2h

With one Mac, the device name is omitted:

> 85% used — runs out in ~2h

## Managing Devices

Go to **Settings → Devices** on iPhone to see all paired Macs and remove ones you no longer use.
