> **TL;DR** — Each platform has a purpose: Mac monitors, iPhone notifies, Watch glances.

## Platform overview

| Platform | Role | Min version |
|----------|------|-------------|
| macOS | Journal monitoring, menu bar UI | macOS 15 |
| iOS | Push notifications, usage dashboard | iOS 18 |
| watchOS | Complications, notification mirror | watchOS 11 |
